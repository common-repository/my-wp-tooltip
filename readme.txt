== Description ==


This plugin will enable tooltip in your Wordpress theme.

Plugin Features

* Very Lightweight.
* Easy to use.
* Customizable. 


== Installation ==

1. Install as a regular WordPress plugin
2. Just use rel attribute in anchor or span. Example code is given below
`<a href="#" title="Tooltip on top" data-rel="tooltip" data-placement="top">Morbi leo risus</a>`  
`<a href="#" title="Tooltip on bottom" data-rel="tooltip" data-placement="bottom">Morbi leo risus</a>`
